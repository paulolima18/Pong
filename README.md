### Getting Started 

- Pong [Implementation of "Pong", the game, in python3 using pygame]
  * Single Player : Single Player mode
  * Multiplayer : Two Player mode

* Run game modes with:

Normal:
```
python3 pong.py
```
2 Players:
```
python3 pong2p.py
```
